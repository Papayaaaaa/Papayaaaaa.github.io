一个假程序员的追星惨案

发表于 2019-04-10

我再也不要随便立flag玩python了QAQ
起因时很久很久之前要给龙龙投一个投票，我特别没耐心so基本重复超过10次的工作基本上我就会开始上网搜代码了orz，后来发现百度盖楼对数据帮助比较大，所以在网上找（抄）了一点自动回帖代码试了下，主要参考的的代码是 魔力Python » 小工具：百度贴吧自动顶贴机不过没有用上cookie函数（我不知道是不是我的打开方式不对我用cookie登录买的小号各种不成功，理论上是可以的well一定是我这条咸鱼的问题）
主要使用的工具包是selenium包，可以使用pip安装pip install -U selenium
PS.需要下载与电脑Chorme的版本相对应的chormedriver，版本一定要对应！！
然后把chormedriver复制到至谷歌浏览器目录（如 C:\Program Files\Google\Chrome\Application）以及当前运行的python根目录（例如我的是D:\Anaconda3）如果不行就把谷歌浏览器目录添加到环境变量path
这里主要参考的是博客 解决chromedriver executable needs to be in PATH问题

代码下载

url.txt——需要盖楼的链接，可以每天修改，现在里面是4月10号的朱一龙贴吧数据楼
reply.txt—回复，每次回复取一条并添加随机字母后缀防止吞楼
tieba.py—python脚本，python版本是3.6

代码改到最后还是用了人工登录，看到有人说这叫人工干预爬虫hhhhhhh，但是确实我买的小号都不好用，人工登录可能还要方便一点，函数login()里面是自动登录首页的代码，但是我后面登录次数太多了需要验证就不太好用了所以主函数里面写的还是需要人工登录一下，这里只放主函数

人工登录时间间隔我设置的是120秒（因为太快了有时候来不及接收验证码），回复时间间隔设置的是300秒，没有试过更快的，怕被封号orz，第一次回复好像有时候会出现滑块验证，如果出现了可以手动帮忙验证下，我懒得继续写滑动验证码了，隔一段时间看一眼就行

——————————————————————————

如果从来没有接触过python的话，可以参考这个网站安装一个python，然后打开python自带的编译器IDLE(Python GUI)，从IDLE–>File–>Open打开python脚本，然后选择菜单Run–>Run Module(当然也可以打开python脚本之后直接F5)

PS. tb上面像又自动顶帖机软件卖。。。可以直接买那个，也不是特别贵，我还没买过但是我感觉别人那个做的比我这个强多了

最后祝大家快乐追星~
